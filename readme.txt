Connect AutomatorWP with WP Ulike

== Description ==

> **Important:** Now all free integrations are included in [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") so you no longer need to install them one by one!
>
> We will continue working to improve AutomatorWP and make it easier, faster and more accessible to everyone.

[WP Ulike](https://wordpress.org/plugins/wp-ulike/ "WP Ulike") is one of the best and fastest ways to add like and dislike functionality to your WordPress website.

This plugin automatically connects [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") with WP Ulike adding new triggers that you can use to connect with other plugins and automate workflows letting you save time and get focused on your most important work.

= Triggers =

* User likes any/specific post.
* User likes a comment of any/specific post.

= Pro version =

[Pro version](https://automatorwp.com/add-ons/wp-ulike/ "AutomatorWP - WP Ulike") available that let's you expand the integration between AutomatorWP and WP Ulike bringing you more ways to automatize your website likes.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
